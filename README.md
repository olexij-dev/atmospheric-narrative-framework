# atmospheric-narrative-framework
a Unity framework for creating slow-paced narrative exploration games with living NPCs, dynamic dialogue, atmospheric environments and event-driven storytelling

Focus areas:

- first-person exploration
- NPC routines
- lightweight interaction
- dialogue events
- ambient sound
- environmental storytelling
- cycle/day progression
- save/load
- mood systems such as rain, fog, flicker, silence

This repository is story-agnostic.

It contains no game-specific lore or commercial IP.
